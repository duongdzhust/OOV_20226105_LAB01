1. What happens if users choose “Cancel”? 
It won’t be saved and launched.
2. How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do” and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help).  
To customize options like “Yes” and “No” or “I do” and “I don’t” using Java, you can use JOptionPane from the javax.swing package. This allows you to create a dialog box with customizable button options.
